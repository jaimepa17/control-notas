# control notas

Repositorio inicial para el proyecto.

## Inicio rapido

1. Clona el repositorio.
2. Crea una rama para tu trabajo.
3. Agrega codigo en `src/` y pruebas en `tests/`.

## Estructura sugerida

- `src/`: codigo fuente
- `tests/`: pruebas
- `docs/`: documentacion

## Convenciones

- Mantener commits pequenos y descriptivos.
- Escribir pruebas para cambios relevantes.
- Documentar decisiones importantes en `docs/`.
